# AI Translator

AI-powered language translation application built using React, TypeScript, Express.js and Google Cloud Translation API.

## Features

- Multi-language translation
- Fast and accurate results
- Modern responsive UI
- Secure API integration
- Full-stack architecture

## Tech Stack

Frontend:
- React
- TypeScript
- Vite

Backend:
- Node.js
- Express.js
- TypeScript

API:
- Google Cloud Translation API

## Installation

```bash
npm install
npm run dev
